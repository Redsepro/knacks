🏁 Objetivos

El contenido de carpetas con imágenes se muestra como miniatura y cuando la vista previa de miniaturas se desactiva, se muestran íconos. Windows solo permite habilitar/deshabilitar las vistas previas de miniatura de ficheros y carpetas en el Explorador usando la función Mostrar siempre íconos, nunca vista en miniatura en el diálogo Opciones de carpeta. No existe una opción para habilitar/deshabilitar la vista previa de miniatura selectivamente solo para carpetas o para ficheros.

Este truco permite habilitar o deshabilitar la miniatura de carpetas, especialmente útil cuando necesite cambiar los íconos de la carpeta.

👨🏽‍🏫 Instrucciones

Desactivar miniaturas de carpetas

  • Descargar el fichero o crear un fichero de texto con la extensión .reg y este contenido:
Windows Registry Editor Version 5.00
; Disable folder thumbnails without disabling thumbnail view for files
; Created by Ramesh Srinivasan (10/0672019) for Winhelponline.com
; Note: This fix does NOT reset your saved folder view settings.
[HKEY_CURRENT_USER\Software\Classes\Local Settings\Software\Microsoft\Windows\Shell\Bags\AllFolders\Shell]
"Logo"="d:\\unexistingfile.jpg"

disable_folder_thumbs.reg

  • En el Explorador de archivos, hacer doble clic en el archivo .reg para insertar la configuración al registro.
  • Confirmar para aplicar la configuración.
  • Limpiar la caché de miniaturas para actualizar las vistas previas de las carpetas. Seguir uno de los métodos descritos en el artículo Cómo borrar y reconstruir el caché en miniatura en Windows.

Activar miniaturas de carpetas

  • Descargar el fichero o crear un fichero de texto con la extensión .reg y este contenido:
Windows Registry Editor Version 5.00
; Enable folder thumbnails.
[HKEY_CURRENT_USER\Software\Classes\Local Settings\Software\Microsoft\Windows\Shell\Bags\AllFolders\Shell]
"Logo"=-

enable_folder_thumbs.reg

  • En el Explorador de archivos, hacer doble clic o ejecutar con permisos de administrador el fichero .reg para insertar la configuración al Registro.
  • Confirmar para aplicar la configuración.
