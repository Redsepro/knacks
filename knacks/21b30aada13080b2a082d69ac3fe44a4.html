🏁 Objetivos

En los clientes de correo Thunderbird o BetterBird, el orden de las cuentas se establece según el orden de creación. Si solo se usa una cuenta, no constituye un problema. Pero si se usa varias cuentas, el desorden puede resultar incómodo. A continuación, se muestran tres modos para cambiar el orden de tus cuentas: desde la interfaz en las versiones 91.0 o superiores, editando el fichero de preferencias o usando un complemento.

👨🏽‍🏫 Instrucciones

Desde la interfaz

  • En el menú de la ventana principal, seleccionar la opción 👤Configuración de cuentas.
  • En la columna izquierda se pueden mover las diferentes cuentas arrastrando y soltando con el cursor al lugar deseado. En algunos sistemas, es necesario mantener presionado Shift o Ctrl + Shift para activar el arrastre.

Editar el fichero de preferencias

  • En el menú de la ventana principal, seleccionar la opción ❓Ayuda > Información para solucionar problemas.
  • En la lista, hacer clic en el botón Abrir carpeta.
  • Encontrar el fichero prefs.js para editarlo con su editor de texto de preferencia.
  • Buscar el texto user_pref("mail.accountmanager.accounts” en donde se encuentran las ID de las cuentas registradas con la plantilla de account[número]. Ordenarlas a su gusto.

Usando un complemento

  • Si no se siente cómodo editando manualmente ficheros del sistema, se puede cambiar el orden de las cuentas de correo directamente usando el complemento Manually Sort Folders (compatible hasta Thunderbird 103.0). Tras instalarlo y reiniciar el cliente si es necesario, acceder a sus opciones y seleccionar la pestaña Sort folders para ordenar las carpetas dentro de cada cuenta.
